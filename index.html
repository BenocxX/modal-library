How to import the librairie?

To use our modal librairie, you first need to import the javascript and css files to your project. 


                            <!-- HTML -->
                            <!DOCTYPE html>
                            <html lang="en">
                            <head>
                                <meta charset="UTF-8">
                                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                <link rel="stylesheet" href="public/style/modals.css">
                                <title>Your website</title>
                            </head>
                        

                            // Javascript
                            import * as md from "./modal.js";

Create an Alert Modal

To create a simple alert modal, you can simply call the function "showAlertModal()". 


                            <!-- HTML -->
                            <button class="myButton bg-light-blue" data-modal-target="alertModal">
                                Button
                            </button>
                        

                            // JavaScript
                            const myButton = document.querySelector(".myButton");
                            myButton.addEventListener("click", () => {
                                md.showAlertButton(); // Will show an alert modal
                            });

Customizing an Alert Modal

You can customize elements shown in the alert modal using the detail object. Every field in this object as a default value, so you don't need to specify the field if you don't need it.

You can also specify a callback function to be called when the user press the button in the modal. 


                            <!-- HTML -->
                            <button class="myButton bg-light-blue" data-modal-target="alertModal">
                                Button
                            </button>
                        

                            // JavaScript
                            const myButton = document.querySelector(".myButton");
                            myButton.addEventListener("click", () => {
                                md.showAlertButton({
                                    title: "Custom Alert Title",
                                    text: "My custom alert description",
                                    position: "top", // Choice: { "top", "middle" }
                                    animationIn: "down", // Choice: { "down", "fade" }
                                    animationOut: "up" // Choice: { "up", "fade" }
                                }, () => {
                                    alert("This is a callback!");
                                });
                            });

Create a Confirm Modal

To create a simple confirm modal, you can simply call the function "showConfirmModal()". 


                            <!-- HTML -->
                            <button class="myButton bg-light-blue" data-modal-target="confirmModal">
                                Button
                            </button>
                        

                            // JavaScript
                            const myButton = document.querySelector(".myButton");
                            myButton.addEventListener("click", () => {
                                showConfirmModal(); // Will show a confirm modal
                            });

Customizing a Confirm Modal

You can customize elements shown in the confirm modal using the detail object. Every field in this object as a default value, so you don't need to specify the field if you don't need it.

You can also specify a callback function taking a boolean as a parameter to be called when the user press one of the two buttons in the modal. 


                            <!-- HTML -->
                            <button class="myButton bg-light-blue" data-modal-target="confirmModal">
                                Button
                            </button>
                        

                            // JavaScript
                            const myButton = document.querySelector(".myButton");
                            myButton.addEventListener("click", () => {
                                showConfirmModal({
                                    title: "Custom Alert Title",
                                    text: "My custom alert description",
                                    confirmButton: "Custom confirm button",
                                    denyButton: "Custom deny button",
                                    position: "top", // Choice: { "top", "middle" }
                                    animationIn: "down", // Choice: { "down", "fade" }
                                    animationOut: "up" // Choice: { "up", "fade" }
                                }, (answer) => {
                                    answer ? alert("User confirmed!") : alert("User denied!");
                                });
                            });

Creating a modal with Ajax

You can also fetch a modal with Ajax! Use a pre-existing html file or a URL to fetch your modal. 


                            // Javascript
                            const myButton = document.querySelector(".myButton");
                            myButton.addEventListener("click", () => {
                                md.showAjaxModal("path/to/html/file or URL");
                            });

Creating a custom modal

You can also create your own custom modal. 


                            <!-- HTML -->
                            <!-- Use the "data-modal-launcher" attribute to bind the button to all launchers -->
                            <!-- Then use the "data-modal-id" attribute to bind the button to your custom modal -->
                            <button class="myButton bg-light-blue" data-modal-launcher
                                        data-modal-id="myModal">
                                Button
                            </button>
                            
                            <!-- You need to specify which animations you want to use with: -->
                            <!--     - data-animation-in="down" (or "fade")  -->
                            <!--     - data-animation-out="up" (or "fade")  -->
                            
                            <!-- You also need to specify where you want your modal to be shown: -->
                            <!--     - data-position="top" (or "middle")  -->
                            <div data-animation-in="down" data-animation-out="up" data-position="top" class="modal" id="exampleModal1" tabindex="-1">
                                <div class="modal-dialog">
                                    <div class="modal-header-container">
                                        <h3 class="modal-title">Title</h3>
                                        <button data-close-button class="modal-button">X</button>
                                    </div>
                                    <div class="modal-content-container">
                                        <p class="modal-text">Text</p>
                                    </div>
                                    <div class="modal-footer-container">
                                        <button data-close-button class="modal-button">Close</button>
                                        <button class="modal-button">Test</button>
                                    </div>
                                </div>
                            </div>
                            
                            <div class="modal-shadow"></div>
                        

                            // JavaScript
                            const myButton = document.querySelector(".myButton");
                            myButton.addEventListener("click", onModalLaunch);

Customizing elements in the modal

You can customize the information shown in your modal with Javascript like you would do for anything on your website. 


                            // JavaScript
                            const modal = document.querySelector("#myModal");
                            modal.addEventListener("modal.shown", function (e) {
                                const dialog = this.querySelector(".modal-dialog");
                                const dialogTitle = dialog.querySelector("h3");
                                const dialogText = dialog.querySelector("p");
                                
                                dialogTitle.innerText = "My custom title";
                                dialogText.innerText = "My custom text";
                            });

Passing info through modal

You can also use the detail object to pass information to your custom modal. 


                            // JavaScript
                            const myButton = document.querySelector("#myButton");
                            myButton.addEventListener("click", (e) => {
                                onModalLaunch(e, {
                                    title: "Where am I from?",
                                    text: "I am text coming from the \"detail\" parameter!"
                                });
                            });
                            
                            const modal = document.querySelector("#myModal");
                            modal.addEventListener("modal.shown", function (e) {
                                const dialog = this.querySelector(".modal-dialog");
                                const dialogTitle = dialog.querySelector("h3");
                                const dialogText = dialog.querySelector("p");
                            
                                dialogTitle.innerText = e.detail.title; // Use the detail object to get your information
                                dialogText.innerText = e.detail.text;
                            });

Using the close custom event

You can use the "modal.close" custom event to run code when the modal is closing. 


                            // Javascript
                            const modal = document.querySelector("#myModal");
                            modal.addEventListener("modal.close", () => {
                                // Your code...
                            })
